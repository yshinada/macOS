# WordPress.com への移し方

このサイトのソースは、まずローカル確認用として作っています。
WordPress.com ではプランによって使えるコードが違うため、最初は固定ページとして内容を移す方法が安全です。

## まずやること

1. WordPress.com にログインする
2. 左メニューから「固定ページ」を開く
3. 「新規固定ページを追加」を押す
4. タイトルを「プログラム公開」または「ソフトウェア」にする
5. 下の「貼り付け用本文」をブロックとして入力する
6. ヒーロー画像 `assets/hero-software-workspace.png` を「メディア」にアップロードする
7. 画像ブロックでヒーロー画像をページ上部に入れる
8. ダウンロード用ファイルをメディアにアップロードして、各リンクを差し替える
9. プレビューで確認してから「公開」を押す

## 不具合報告・問い合わせ

使っているOS、プログラム名、バージョン、発生した状況を添えてご連絡ください。再現手順があると確認しやすくなります。

## ダウンロードリンクの作り方

1. WordPress.com の左メニューで「メディア」を開く
2. DMG / ZIP / EXE などの配布ファイルをアップロードする
3. アップロードしたファイルを開いてURLをコピーする
4. 固定ページの「ダウンロード」の文字にリンクを設定する

## 注意

- 無料プランでは、自由なCSSやJavaScriptを使ったデザイン移植は制限されることがあります。
- まずはWordPressのブロックだけで公開し、あとで必要に応じて見た目を整えるのがおすすめです。
- `script.js` のような動的なフィルター機能は、WordPress.com のプランによって使えない場合があります。
